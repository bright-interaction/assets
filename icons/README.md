# Icons

Brand and service icons for Bright Interaction automation projects.

## Usage

### CDN URL (recommended)
```
https://cdn.jsdelivr.net/gh/bright-interaction/assets/icons/[filename]
```

### Raw URL (alternative)
```
https://raw.githubusercontent.com/bright-interaction/assets/main/icons/[filename]
```

## Available Icons
- logo.svg
- automation.svg
- crm.svg

## Examples
```
https://cdn.jsdelivr.net/gh/bright-interaction/assets/icons/logo.svg
https://cdn.jsdelivr.net/gh/bright-interaction/assets/icons/automation.svg
https://cdn.jsdelivr.net/gh/bright-interaction/assets/icons/crm.svg
```
